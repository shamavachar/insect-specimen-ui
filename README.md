# Insect Specimen Landing Page - Assignment 2

## Overview
This project is an elegant, split-screen landing page designed to showcase an insect specimen (**Pachyteria equestris**). Created as part of the Cohort 3.0 curriculum (Assignment 2), it features a highly detailed layout. The left side presents scientific classification data, while the right side displays specific specimen details like locality, sex, size, and pricing.

This project was built and styled based on the reference design provided in `assets/image (3).png`.

## Features
- **Split Layout Design:** A clean, dual-section layout balancing a light left panel for taxonomy with a bold red (`#B32E27`) right panel for specimen details.
- **Custom Typography:** Utilizes Google Fonts **"Inter"** for clean UI text and **"Cormorant Garamond"** for elegant headings and vertical text.
- **Rich Iconography:** Integrates **Remix Icon** for intuitive visual cues (search, location, gender, ruler).
- **Interactive Elements:** Features custom CSS toggle switches and hover effects on navigation links.
- **Centered Image Showcase:** Uses absolute positioning to center the primary insect image over the split background, creating a seamless visual flow between the two sections.
- **SEO & Meta Tags:** Includes Open Graph and Twitter Card meta tags for better social sharing previews.

## Tech Stack
- **HTML5:** Semantic HTML structure and metadata.
- **CSS3:** Custom styling, Flexbox, absolute/relative positioning, custom toggles, and imports.
